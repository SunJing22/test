分层组件

Daisy-Dashboard module

Daisy-Dashboard提供界面操作，配置和部署相关的所有操作都在界面上进行

Daisy-API module

其中Daisy-API位于控制层，提供符合RestFul标准的接口

Daisy-Deployment module

位于执行层，负责执行具体的部署工作。Daisy-Deployment模块,使用了动态插件架构，支持不同的后端部署。 当前支持的后端列表包括：OpenStack、SDN和CEPH。后续可以很方便的扩展支持其他的后端。

key features

hardware discovery

Automating Distributed Systems Deployment to Baremetal

support Deployment and upgrade for multiple OpenStack distributions

support Deployment and upgrade for SDN

support Deployment and upgrade for Ceph

support for template function

Daisy可以将一个OpenStack环境的组网配置导出为模板，在部署一个相似的OpenStack环境时，只需要导入模板即可完成配置。

support for auto scale

对于需要做节点扩容的OpenStack环境，Daisy支持节点上电自动完成部署，自动加入OpenStack集群。
