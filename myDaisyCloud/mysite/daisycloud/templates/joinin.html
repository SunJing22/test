How to join in us:

Preparatory work

1. Account setup           ....

First of all, you will need a Launchpad account.

Join The OpenStack Foundation, remeber use the same email address as step one.

Visit https://review.openstack.org/ and click the "Sign In",Log in with your Launchpad ID.The first time you login OpenStack's Gerrit, you will nee to select a unique username.

Agree to the Individual Contributor License Agreement and provide contact information.

2. Install git and config git          ....

Install git

For Centos: yum install git

For Ubuntu: apt-get install git

git config

git config --global user.name "Firstname Lastname"

git config --global user.email "your_email@youremail.com"

3. Installing git-review          ....

For centos : apt-get install git-review

For Ubuntu : yum install git-review

Begin work with Daisy

1. Clone code          ....

git clone https://git.openstack.org/openstack/daisycloud-core

git config --global gitreview.username yourgerritusername

In case you had already tried to setup git-review and it failed, it might be necessary to remove the Gerrit remote from git:

git remote rm gerrit

2. Starting a change          ....

Starting a change details.

work on bugs,modify code locally.

3. Committing a change          ....

Committing a change details.

git commit -am "change messages"

4. Submitting a change for review          ....

git review
